
# 🧠📊 AI Mental Model Generator

An AI-powered application that helps users **think better, make smarter decisions, and understand complex topics** using structured mental models and decision frameworks.

---

## 💡 Overview

Instead of giving direct answers, this tool teaches **how to think** by generating powerful mental models used in real-world decision-making.

---

## 🚀 Features

### 🧠 Mental Model Generator
- Generate 3–5 mental models for any topic
- Includes:
  - Model name
  - Explanation
  - Real-life example
  - When to apply

---

### ⚖️ Decision Framework
- Input a real-life problem
- AI applies relevant mental models
- Provides structured decision guidance

---

### 🎯 Multiple Modes
- **Standard Mode** → Deep insights  
- **Simple Mode** → Beginner-friendly explanations  

---

### 💾 Save Models
- Store useful mental models
- Build your own thinking library

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- Session State (for storing models)

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/ai-mental-model-generator.git
cd ai-mental-model-generator
````

---

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file in the root directory:

```env
GOOGLE_API_KEY=your_gemini_api_key
```

Get your API key from:
[https://makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)

---

### 4️⃣ Run the Application

```bash
streamlit run app.py
```

---

## 🧪 How It Works

### Generate Mental Models

1. Enter a topic (e.g., “money”, “career”, “decision-making”)
2. Choose explanation mode
3. AI generates structured thinking frameworks

---

### Apply to Real Problems

1. Enter your problem
2. AI applies relevant mental models
3. Get actionable insights and decision guidance
